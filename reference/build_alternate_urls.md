# Build alternate URLs to try if primary URL fails

Build alternate URLs to try if primary URL fails

## Usage

``` r
build_alternate_urls(end_year, school_year, level, data_type, count_day)
```

## Arguments

- end_year:

  School year end

- school_year:

  School year string

- level:

  "school" or "district"

- data_type:

  "grade" or "demo"

- count_day:

  "45", "135", or "180"

## Value

Vector of alternate URLs
