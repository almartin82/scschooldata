# Package index

## Fetch Enrollment Data

Functions for downloading enrollment data from SCDE

- [`fetch_enr()`](https://almartin82.github.io/scschooldata/reference/fetch_enr.md)
  : Fetch South Carolina enrollment data
- [`fetch_enr_multi()`](https://almartin82.github.io/scschooldata/reference/fetch_enr_multi.md)
  : Fetch enrollment data for multiple years
- [`fetch_report_cards()`](https://almartin82.github.io/scschooldata/reference/fetch_report_cards.md)
  : Fetch enrollment data from SC Report Cards
- [`get_available_years()`](https://almartin82.github.io/scschooldata/reference/get_available_years.md)
  : Get list of available years for SC enrollment data
