# Download a single headcount file from SCDE

Download a single headcount file from SCDE

## Usage

``` r
download_headcount_file(end_year, school_year, level, data_type, count_day)
```

## Arguments

- end_year:

  School year end

- school_year:

  School year string (e.g., "2024-25")

- level:

  "school" or "district"

- data_type:

  "grade" or "demo"

- count_day:

  "45", "135", or "180"

## Value

Data frame
